LHV blacklist name matching

What works:

  • Detects following cases:
    • "Osama Bin Laden"
    • "Bin Laden, Osama"
    • "Laden Osama Bin"
    • "to the osama bin laden"
    • "osama and bin laden"
    • "osama,,bin,,,,laden"
    • "OsAmA BiN LaDen"
    • Additionally: "Bin Laden" and other partial searches

How to use:

  1. Insert name into the field
  2. Check for exact matches
  3. Check for matches with noise words in a list below
  4. Check for partial matches
  5. Check for names that definitely are not in the list

What was used:

  • Javascript
  • Jquery
  • Bootstrap
  • Bootstrap One Page Wonder Theme
  • Random generated list of names
  • Noise words
  • My friend google
  • Binary search for name search when looking for exact matches
  • Regex search (for loop over all elements) for partial matches
